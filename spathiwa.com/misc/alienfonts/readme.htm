Star Control 2 - Alien Fonts Package
by Ringwraith

This package contains fonts for Windows - in Bitmap (.FON) format - copied from the fonts used for conversations with the various aliens in Star Control 2.  The bitmap format means that they can't be used in programs such as Photoshop, as far as I know, but can be used in Windows Paint, for example (oh well; there's always the good old copy-paste).  Perhaps something can be done to convert them to Truetype, but I don't know how good that will look.

For another font package by me, containing other Star Control 2 fonts for Windows (and for every other imaginable thing related to SC2) - visit The Pages Of Now And Forever - All About Star Control.

Some things to contsider while creating graphics using these fonts:

Spacing - I tried to preserve the spacing used in the game (between letters, words and lines) as accurately as possible; however, some things will come out wrong, for example the spacing between two characters in the game depends on which letters they are, and so on so forth.

Scale - these fonts are scaled 200% - each pixel in the game is a square of 2x2 in these fonts.  This is for the simple reason that the graphics used in the game, in a screen resolution of 300x200, will appear much too small with the resolutions most poeple use today. If you utterly and absolutely must have the original pixel-for-pixel scale, for whatever reason, use these fonts and just downscale the resulting image by 50%, using whatever graphics proggie you find.

Visibility - in the game, for visibility reasons, the font is always white, and always has a one-pixel-wide border of black around it; so if you want to create an authentic-looking image you should worry about creating such a border yourself:

Usefulness - yes, I am 100% aware that having wasted any considerable amount of time on creating something like this package is bordering on the utterly ridiculous, so no need to bring that to my attention, thank you very much.
